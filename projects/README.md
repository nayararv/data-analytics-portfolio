# 📊 SQL Sales Analysis

Projeto de análise de dados utilizando **SQL** para explorar e gerar insights sobre o desempenho de vendas de uma empresa fictícia.

O objetivo é demonstrar habilidades em:

- Modelagem e criação de banco de dados
- Escrita de consultas SQL
- Análise de métricas de negócio
- Extração de insights a partir de dados

---

# 🎯 Problema de Negócio

A empresa deseja entender melhor o desempenho de suas vendas para responder perguntas como:

- Quais produtos vendem mais?
- Quais vendedores possuem melhor desempenho?
- Qual região gera maior faturamento?
- Como as vendas evoluem ao longo do tempo?

Para isso foi construído um banco de dados e um conjunto de consultas analíticas em SQL.

---

# 🗄 Estrutura do Banco de Dados

O banco de dados simula um cenário comercial com as seguintes entidades:

- **clientes**
- **produtos**
- **vendedores**
- **vendas**
- **itens_venda**

Essas tabelas permitem analisar:

- faturamento
- volume de vendas
- performance de vendedores
- ranking de produtos
- evolução temporal das vendas

# 🔎 Análises Realizadas

As consultas desenvolvidas neste projeto permitem responder perguntas de negócio como:

### 📦 Produtos
- Produtos mais vendidos
- Produtos com maior faturamento

### 👩‍💼 Vendedores
- Ranking de vendedores por faturamento
- Volume de vendas por vendedor

### 👥 Clientes
- Clientes com maior volume de compras
- Frequência de compras por cliente

### 📈 Vendas
- Evolução das vendas ao longo do tempo
- Ticket médio
- Faturamento total

### 🏆 Ranking
- Top produtos
- Top vendedores
- Regiões com maior faturamento
