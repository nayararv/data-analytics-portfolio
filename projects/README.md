# 📂 Data Analytics Projects

Esta pasta reúne projetos desenvolvidos para demonstrar habilidades em **SQL, análise de dados e visualização com Power BI**.

Cada projeto aborda um problema de negócio e apresenta a análise realizada.

---

## 📊 Power BI Sales Dashboard

Dashboard estratégico desenvolvido em **Power BI** para análise de desempenho de vendas.

Principais análises:
- Faturamento total
- Ticket médio
- Performance de vendedores
- Vendas por região e categoria

🔗 Acesse o projeto:  
../powerbi-sales-dashboard

## 🗄 SQL Sales Analysis

Projeto de análise de dados utilizando **SQL** para explorar métricas de vendas.

Principais análises:

- Ranking de vendedores
- Evolução das vendas
- Métricas de faturamento
