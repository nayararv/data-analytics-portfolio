# 📁 Projetos

## 📊 Dashboard de Vendas — Power BI

🔗 Repositório
https://github.com/nayararv/powerbi-dash-vendas-comercial-

Projeto de **Business Intelligence desenvolvido em Power BI** para análise de desempenho comercial.

O dashboard analisa dados de vendas simulados com foco em **indicadores estratégicos de negócio**, permitindo acompanhar a performance comercial da empresa.

### Principais análises

* Faturamento total
* Quantidade vendida
* Ticket médio
* Desempenho de vendedores
* Comparação entre vendas e metas

### Modelo de dados

O projeto utiliza **modelagem dimensional (Star Schema)** composta por:

Tabela fato:

* vendas

Tabelas dimensão:

* clientes
* produtos
* vendedores
* datas

Tabela de apoio:

* metas de vendas

### Tecnologias utilizadas

* Power BI
* DAX
* SQL (MySQL)

## 🗄 SQL — Análise de Vendas

🔗 Repositório
https://github.com/nayararv/sql-analise-de-vendas

Projeto de **análise de dados utilizando SQL**, simulando um banco de dados de vendas de uma empresa.

O objetivo do projeto é demonstrar habilidades em:

* modelagem de banco de dados
* consultas SQL
* análise de métricas de negócio

### Estrutura do banco

O banco de dados contém tabelas que representam o processo de vendas:

* clientes
* produtos
* vendedores
* vendas
* datas

### Análises realizadas

O projeto contém **50 consultas SQL**, incluindo:

📦 Produtos

* produtos mais vendidos
* ranking de produtos
* produtos mais lucrativos

👥 Clientes

* clientes que mais compram
* clientes por localização
* segmentação de clientes

💰 Vendas

* faturamento total
* ticket médio
* quantidade de vendas

🧑‍💼 Vendedores

* ranking de vendedores
* vendedores com maior faturamento
* performance de vendas

📅 Tempo

* faturamento mensal
* tendência de vendas
* evolução de vendas no tempo

### Tecnologias utilizadas

* SQL
* MySQL
* Git
* GitHub
